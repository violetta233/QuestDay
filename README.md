# QuestDay
QuestDay - это увлекательное и мотивирующее мобильное приложение для выработки полезных привычек, которое сочетает в себе классический трекер привычек с элементами ухода за виртуальным питомцем для повышения мотивации и вовлеченности.
HEAD

**Цель проекта:**

Разработать кроссплатформенное приложение, которое помогает формировать полезные привычки в игровой форме, объединяя трекинг прогресса и увлекательные элементы гейминга.
HEAD

**Задачи проекта:**

 1. Спроектировать архитектуру приложения и определить ключевые модули системы.
 2. Реализовать основные функции планировщика дня и трекера привычек.
 3. Разработать дизайн и анимации виртуального питомца — кролика.
 4. Создать музыкальное оформление и звуковые эффекты.
 5. Провести оптимизацию и пользовательское тестирование

Стек технологий:
 1. С#
 2. Unity
